    skyADLib

    Altium Designer 设计库主要包括Schlib,Pcblib,IntLib,Pad Via library,Database library,SVN Database library,Database Link 等库文件。

    1. Schematic library

    (1) 常用贴片电阻、（陶瓷、钽电容）电容、电感继承自-嘉立创元器件库: https://gitee.com/JLC_SMT/JLCSMT_LIB

    (2) 直插（水泥）电阻、电解电容、功率电感

    (3) SMT连接器

    (4) DIP连接器

    (5) 测试点、测试针

    (6) 变压器

    (7) 继电器

    (8) 集成电路：

    • 稳压器（三端稳压器、五端稳压器、固定稳压器、可调稳压器、开关集成稳压器）
    • DC/DC变换器
    • 运算放大器、厚膜电路（电源厚膜电路、功放厚膜电路）
    • 数字逻辑电路（TTL集成电路、CMOS集成电路）
    • A/D、D/A转换电路
    • 音频电路
    • 视频
    • 专用电路
    • （电视机用集成电路、遥控用集成电路、音响用集成电路、影碟机用集成电路）
    • MCU
    • DSP
    • FPGA
    • 时钟和计时电路
    • 接口电路
    • UART
    • CAN
    • IIC
    • USB
    • EtherNet
    • 隔离器件
    • 电机驱动器
    • 电源管理
    • 开关和多路复用器件
    • 射频和微波
    • 无线连接（蓝牙、Wifi、ZigBee）

    (9) 二极管：

    • 普通二极管（整流、检波二极管）、
    • 单结晶体管、
    • 专用二极管（双向触发二极管、快恢复二极管、变容二极管、开关二极管）

    (10) 三极管：

    • 普通三极管（低频三极管、高频三极管、大功率三极管、开关三极管、达林顿管）、
    • 场效应管（结型场效应管、绝缘栅型场效应管）、
    • 晶闸管

    (11) 开关

    (12) 保险元件：保险管、可恢复熔丝、熔断电阻器

    (13) 石英晶体与陶瓷器件：

    • 石英晶体谐振器件(有源、无源)
    • 陶瓷滤波器与陷波器
    • 声表面波器件

    (14) 发光器件：

    • 发光二极管（普通发光二极管、电压型发光二极管、闪烁型发光二极管）、
    • 数码管
    • 氖管
    • 指示灯
    • 显像管
    • 液晶显示器
    • 等离子显示器

    (15) 片状器件：片状电阻器、片状电容器、片状电感器、片状二极管、片状三极管

    (16) 电声器件：

    • 扬声器（低音扬声器、高音扬声器、全频带扬声器、号筒扬声器）、
    • 耳机（电磁式耳机、耳塞、动圈式耳机）、
    • 蜂鸣器（压电式蜂鸣器、电磁式蜂鸣器）、
    • 传声器（动圈式传声器、电容式传声器、驻极体扬声器）

    (17) 敏感器件：

    • 温敏器件（热敏电阻器、温敏二极管、铂电阻元件）
    • 光敏器件（硅光电池、光敏二极管、光敏三极管）
    • 湿敏器件
    • 力敏器件
    • 磁敏器件
    • 气敏器件
    • 电压敏感器件（压敏电阻器、瞬态电压抑制二极管）

    (18) 传感器：

    • 温度传感器
    • 紫外线传感器
    • 光电耦合器
    • 应变式力传感器
    • 电感式接近传感器
    • 磁补偿式电流传感器
    • 霍尔传感器
    • 热释电红外传感器
    • 红外光电开关

    4. Pad Via library

    常用的标准化过孔，主要分三类： (1) 信号过孔 —— 要求过孔结构对信号影响最小 (2) 电源地过孔 —— 要求过孔的分布电感最小 (3) 散热过孔 —— 要求过孔的热阻最小
